StaySense

Smart Hotel Pricing Analytics for Everyone

StaySense is a Data Analytics and Machine Learning-based system designed to evaluate hotel price fairness on OYO. It helps customers identify whether a hotel is overpriced or worth the cost based on key factors like city, category, rating, facilities, and discount percentage.

View Dashboard Download Report

About the Project

StaySense addresses the challenge of price fairness in the hotel industry by leveraging data analytics and machine learning to evaluate OYO hotel pricing strategies.

Problem

Customers often struggle to determine if they're paying a fair price for hotel accommodations, while hotel chains like OYO need better insights into their pricing strategies.

Solution

StaySense analyzes multiple factors to determine price fairness, helping customers make informed decisions and OYO optimize their pricing strategies.

What I Did

  • Web Scraping: Collected 3,800+ hotel listings from OYO website
  • SQL Database Management: Stored and cleaned raw hotel data using MySQL
  • Advanced Data Cleaning & Feature Engineering using Pandas & Numpy
  • Exploratory Data Analysis (EDA)
  • Outlier & Missing Value Treatment
  • Segmentation & Business Insight Extraction
  • Power BI Dashboard Development

Project Workflow

Web Scraping

Collected 3,800+ hotel listings from OYO website with detailed information on pricing, facilities, ratings, and locations.

SQL Database Management

Stored and cleaned raw hotel data using MySQL, ensuring data integrity and efficient querying capabilities.

Data Cleaning & Feature Engineering

Performed advanced data processing in Google Colab using Pandas & Numpy, handling encoding issues and irregular text formats.

Exploratory Data Analysis

Conducted univariate, bivariate, and multivariate analysis to uncover patterns and relationships in the data.

Outlier & Missing Value Treatment

Applied logical imputation using city-wise median strategy to handle missing values and outliers effectively.

Segmentation & Business Insights

Created price bands, facility buckets, and analyzed membership impact to extract actionable business insights.

Power BI Dashboard

Built interactive analytics dashboard for both Customers and OYO Strategy Teams with advanced filtering capabilities.

Challenges & Solutions

Encoding & Unicode Issues

Encountered encoding and unicode problems in scraped price data which required specialized text processing techniques to resolve.

Irregular Text Formats

Facility and rating columns contained inconsistent text formats that needed standardization through advanced string manipulation.

Missing Value Logic Decisions

Evaluated multiple strategies (mean vs median vs segmentation) for handling missing values, ultimately implementing city-wise median imputation.

Power BI Filtering Complexity

Overcame challenges with Power BI filtering logic, drillthrough functionality, and custom measures to create an intuitive user experience.
