About This Project

FinTrack360 is a personal finance analytics project based on my real bank statements. It covers end-to-end data transformation using SQL, Python, and Tableau to discover patterns in income, expenses, savings, and spending behavior.

This project demonstrates my ability to work with real-world financial data, implement complex data transformations, and create actionable business insights from raw transactional data.
